
# Default functions

### import
- Takes one argument, which should be a string filepath. Imports the code and runs it here.

### +
- Add two numbers together. If one is a float, the result will be a float.

### -
- Subtract two numbers. If one is a float, the result will be a float.

### *
- Multiply two numbers. If one is a float, the result will be a float.

### /
- Divide two numbers. Always returns a float.

### %
- Modulo on two integers.

### and
- Logic "and" on two booleans.

### or
- Logic "or" on two booleans.

### not
- Logic "not" on one boolean.

### print
- Print the value of the argument to the console. Argument type does not matter.
